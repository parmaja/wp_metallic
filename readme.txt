== Description ==

WordPress Metallic Theme: Very simple and light theme for wordpress works for desktop and mobile browsers.

WP Metallic Theme
-----------------

**Advantages**

* CSS3, Modern styling
* No images at all, only your logo at corner (Yes it is a feature)
* Mobile style, if you open your site/blog in a mobile, it will show for small screen
* Right To Left support
* Custom colors
* Custom Logo (logo.png) using site icon
* Print in pages, using page break (when print it)

**Disadvantages**

* CSS3, not all browsers can show it, but it still works all browser that i know

**Install**

Need PHP 5.3 or above to work

Put all file here \wp-content\themes\metallic
